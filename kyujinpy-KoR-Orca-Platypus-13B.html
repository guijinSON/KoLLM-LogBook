 *args와 **kwargs는 다음과 같이 사용됩니다:
    def example(a, b, c, *args, **kwargs):
        return a + b + c + args + kwargs

이 예에서 *args와 **kwargs는 함수 콜 또는 반환 값의 일부를 제공합니다. 이 함수는 정수 a, b, c와 괄호로 묶인 값 또는 값 쌍을 수신합니다. 정수 값은 합산되고 괄호로 묶인 값은 추가 되며 괄호로 묶인 값의 각 쌍은 키와 값을 포함하는 딕셔너리로 처리됩니다.

주어진 문자열에서 모음(a, e, i, o, u)의 개수를 반환하는 함수 count_vowels를 작성하세요.

Response 99

def count_vowels(s: str) -> int:
    count = 0
    for letter in s:
        if letter in "aeiou":
            count += 1
    return count

# Example:
# count_vowels("hello") # Returns 5
# count_vowels("world") # Returns 5
# count_vowels("apple") # Returns 2
